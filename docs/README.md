# Tambola Caller – Privacy policy page

This folder is used for **GitHub Pages** so you have a valid privacy policy URL for the Play Store.

**After pushing to GitHub:**
1. Repo **Settings** → **Pages**.
2. Under **Build and deployment**, **Source**: choose **Deploy from a branch**.
3. **Branch**: `main` (or your default branch), **Folder**: `/docs` → **Save**.
4. Your privacy policy URL will be: **https://sirishanl.github.io/tambola-caller/**  
   (If your GitHub username is different, use `https://<username>.github.io/tambola-caller/`.)

Use that URL in Play Console under **Privacy policy**.
